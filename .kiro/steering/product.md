# Product: siyuan-homepage

A plugin for [SiYuan Note](https://github.com/siyuan-note/siyuan) that provides a highly customizable homepage with a rich widget system.

## Core Purpose
Replaces the default SiYuan startup screen with a personalized dashboard. All plugin data is stored inside SiYuan documents — uninstalling the plugin has no side effects on the user's notes.

## Key Features
- Drag-and-drop widget grid layout (desktop and mobile)
- 30+ built-in widgets: task manager, SQL query, heatmap, countdown, weather, music player, Pomodoro timer, quick notes, favorites, recent docs, charts (ECharts), sticky notes, web view, and more
- Top banner with customizable background image
- Visual effects: falling particles, custom mouse cursors
- Sidebar dock panel
- Template center and empty-doc cleaner utilities
- VIP/license system gating advanced features
- Full i18n support (zh_CN / en_US)
- Supports desktop, mobile, browser-desktop, browser-mobile, and desktop-window frontends

## Target Users
SiYuan Note users who want a productive, visually rich home screen for their knowledge base.
